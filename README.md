# Tic Tac Toe

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Purpose
This project was created as a practice exercise while learning:
- JavaScript DOM Manipulation
- Event Handling
- Game Logic
- CSS Styling
- Responsive Web Design

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## Learning Outcome
This project helped me understand how to manipulate the DOM, respond to user interactions, update UI dynamically, and implement game logic using JavaScript.
